# Magno-Relativity — Pulsar Validation (GR vs MR, 25×)

**Release Date:** November 4, 2025  
**Tag:** v2025-11-04

This repository hosts the **public kit** and the pointers to the **NDA-gated private validation package** validating that **Magno-Relativity (MR) reduces mean χ² residuals by ~25× vs General Relativity (GR)** across **98 pulsar systems (ATNF)** with **no tuning parameters**.
